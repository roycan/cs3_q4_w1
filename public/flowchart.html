The Zodiac Explorer is a web application that helps users discover their zodiac sign based on their birth date. The application maintains a record of user submissions and displays zodiac information for all registered users.

Application Flow

flowchart TD A[Start] --> B{Is user registered?} B -->|Yes| C[Show zodiac info] B -->|No| D[Show registration form] D --> E[Submit form] E --> F[Calculate zodiac sign] F --> C C --> G[End]

Detailed Flow Explanation

The application begins at the homepage where it checks if the user is registered. For registered users, the system directly displays their zodiac information along with details about their sign's characteristics, compatibility, and other relevant astrological information. For new users, a registration form is presented where they can enter their name and birth date.

When a new user submits the registration form, the system processes their birth date to calculate their zodiac sign using precise date ranges for each astrological sign. After calculation, the user is redirected to the information page where they can see their zodiac details along with information about other registered users, creating a community-driven experience.
